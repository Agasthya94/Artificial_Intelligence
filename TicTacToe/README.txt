Unbeatable Tic Tac Toe Game - Project

Author 1: Agasthya Vidyanath Rao Peggerla

Project: Game Playing.

==============================================================================================

Required Dependencies:

1. Java compiler. (Jdk 1.7)
2. Eclipse IDE 

===============================================================================================

Folder Structure (GamePlaying.zip):

1. Folder GamePlaying, contains the source code files.
2. README.txt
3. TicTacToeGame.jar
4. Game.bat


================================================================================================

How to use the zip folder:

1. Unzip the folder.
2. Import the project (GamePlaying) into the eclipse.
3. Open TicTacToeGame.java
4. Run the program (Ctrl+F11) 
5. Select the size of the board, and then select who goes first and play the game.

================================================================================================

How to use the jar file.
In the command prompt:
1. java -jar TicTacToeGame.jar
2. Follow the step 5 in the above instructions.

=================================================================================================

How to use the bat file.

1. Run the Game.bat file
2. Follow the instructions as in the previous instructions to play the game.
